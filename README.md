# Peachtree Ledger Website

Static website for Peachtree Ledger (payroll and bookkeeping services).

## Hosting
Hosted on GitHub Pages using static HTML and CSS.

## Structure
- /docs contains the site files
- /assets for CSS and images

## Updating the site
Edit HTML/CSS files and push to main.
Changes deploy automatically via GitHub Pages.

## Forms
Contact form uses Formspree (client-side only).
